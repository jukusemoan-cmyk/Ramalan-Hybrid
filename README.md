## 💰 Model Monetisasi

Proyek ini menggunakan sistem **Token** dengan 3 paket:

| Paket | Harga | Token | Harga per Token |
|:---|:---:|:---:|:---:|
| **Paket Coba** | Rp5.000 | 3 Token | Rp1.666 |
| **Paket Hemat** | Rp20.000 | 15 Token | Rp1.333 |
| **Paket Sultan** | Rp50.000 | 50 Token | Rp1.000 |

- **1 Token = 1x Buka Ramalan Premium** (Asmara, Karir, Keuangan, Kesehatan + Analisis Kepribadian)
- Token **tidak kadaluarsa**.
- Pembayaran via Midtrans (QRIS, GoPay, OVO, DANA, Transfer Bank).
