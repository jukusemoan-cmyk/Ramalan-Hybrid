Kebijakan Privasi

Terakhir diperbarui: 19 April 2026

INTI PRIVASI KAMI: Kami tidak mengumpulkan data pribadi Anda ke server pusat. Semua data Anda disimpan secara lokal di perangkat Anda sendiri.

1. Informasi yang Kami Kumpulkan

Kami TIDAK mengumpulkan atau menyimpan data pribadi Anda di server kami. Semua data yang Anda masukkan di Website ini, termasuk namun tidak terbatas pada:

  • Nama panggilan
  • Nama lengkap (untuk analisis Numerologi)
  • Tanggal lahir (untuk Zodiak, Shio, Weton, Wuku, BaZi)
  • Jam lahir (untuk analisis BaZi - opsional)
  • Nama pasangan (untuk fitur kecocokan)
  • Status token dan riwayat penggunaan

Data tersebut hanya disimpan di dalam browser perangkat Anda sendiri melalui teknologi localStorage. Kami tidak memiliki akses ke data tersebut.

2. Bagaimana Kami Menggunakan Informasi Anda

Karena data disimpan di perangkat Anda sendiri, penggunaan data tersebut sepenuhnya terjadi di perangkat Anda untuk:

  • Mengingat nama panggilan Anda untuk pengalaman yang lebih personal.
  • Melakukan perhitungan Numerologi dari nama lengkap Anda.
  • Melakukan perhitungan BaZi (Empat Pilar Takdir) dari tanggal dan jam lahir Anda.
  • Menyimpan saldo token dan status premium Anda.
  • Menghasilkan URL "bagikan" (shareable link) yang berisi data terenkripsi di URL itu sendiri.

Kami tidak menggunakan data Anda untuk tujuan iklan, profiling, atau dijual kepada pihak ketiga.

3. Layanan Pihak Ketiga

Website ini menggunakan layanan pihak ketiga untuk fungsi tertentu:

  • Midtrans (Payment Gateway): Saat Anda membeli token, Anda akan diarahkan ke halaman pembayaran Midtrans. Data yang Anda berikan di halaman Midtrans tunduk pada Kebijakan Privasi Midtrans.
  • Cloudflare Workers (Backend): Kami menggunakan Cloudflare Workers untuk memproses permintaan pembayaran secara aman. Data transaksi diproses sementara dan tidak disimpan secara permanen.
  • GitHub Pages (Hosting): Website ini di-host di GitHub Pages. GitHub dapat mengumpulkan data log standar. Lihat Kebijakan Privasi GitHub.

4. Tautan ke Situs Lain

Website ini mungkin berisi tautan ke situs web lain. Kami tidak bertanggung jawab atas konten atau praktik privasi situs pihak ketiga tersebut.

5. Keamanan Data

Karena data Anda disimpan secara lokal di perangkat Anda, keamanannya bergantung pada keamanan perangkat dan browser Anda sendiri. Kami menyarankan Anda untuk tidak membagikan perangkat Anda kepada orang yang tidak dikenal.

6. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan berlaku segera setelah diposting di halaman ini.

7. Kontak

Jika Anda memiliki pertanyaan tentang Kebijakan Privasi ini, silakan hubungi kami di: halo@oraclenusantara.id.
